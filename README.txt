=========================================
WPPS 출하통보 자동화 봇 (직원용)
=========================================

[설치 및 실행 방법]

1. 파이썬(Python) 다운로드 및 설치
   - 아직 컴퓨터에 파이썬이 없다면 아래 링크에서 다운로드해 설치해주세요.
   - 다운로드 링크: https://www.python.org/downloads/
   - (중요) 윈도우 설치 시 반드시 첫 화면에서 "Add Python to PATH" 체크박스를 체크해주세요!

2. WPPS 자동화 봇 실행하기
   - 윈도우(Windows) 사용자: 'run_windows.bat' 파일을 더블클릭 하세요.
      * "PC 보호" 파란색 창이 뜨면 -> [추가 정보] 클릭 -> [실행] 클릭
   - 맥(Mac) 사용자: 'run_mac.command' 파일을 더블클릭 하세요.

3. 최초 실행 안내
   - 처음 실행할 때만 필수 프로그램(브라우저 엔진 등)을 다운로드 하느라 1~3분 정도 까만 창이 켜져 있을 수 있습니다. 절대 끄지 말고 기다려주세요!
   - 다운로드가 끝나면 자동으로 크롬 창이 열리고 로지스올(WPPS) 이동 및 자동 입력이 시작됩니다.

※ 주의: '.env' 파일은 아이디/비밀번호가 들어있는 금고 파일입니다. 절대 지우지 마세요.
※ 메일 보낼때 안내: "이 압축파일을 바탕화면 등에 풀고, 본인 OS에 맞는 실행 버튼을 더블클릭 하세요"
